# cloud-js

Personal cloud project in javascript





Comandos para instalação

>npm init -y

>npm install --save-dev typescript @types/node //instalar para o typescript para dev

>npm install express tsx //microframe para rotas

>npm install --save-dev @types/express //types para o express

>npx tsc --init // gerar os arquivos do typescript

>npm install cors dotenv // cors serve para aceitar req de qualquer ip e dotenv e para variavel ambiente

>npm i --save-dev @types/cors // type do cors
