# cloud-js

Personal cloud project in javascript





Comandos para instalação

>npm init -y

>npm install --save-dev typescript @types/node //instalar para o typescript para dev

>npm install express tsx //microframe para rotas

>npm install --save-dev @types/express //types para o express

>npx tsc --init // gerar os arquivos do typescript

>npm install cors dotenv // cors serve para aceitar req de qualquer ip e dotenv e para variavel ambiente

>npm i --save-dev @types/cors // type do cors

>npm install prisma @types/node --save-dev // instalar o gerenciador do banco de dados

>npm install @prisma/client @prisma/adapter-pg dotenv // instalar dependencia do gerenciador do banco de dados

>npx prisma init // para criar o arquivo prisma.config e schemas

>npx prisma migrate dev// criar o schema no banco

>npx prisma generate // forcar gerar a pasta generate

>npm install zod

>npm install bcryptjs // biblioteca para criptografar senha

>npm install jsonwebtoken // para gerar token ao logar

>npm i --save-dev @types/jsonwebtoken // tipagem para jsonwebtoken







//comandos para docker

>git clone https://github.com/FabinhoXXD/cloud-js.git // clona repositorio
>docker compose up -d --build // dentro da pasta monta o container

>git pull // dentro da pasta do projeto puxa update do GitHub
